# UI 디자인 가이드

## 디자인 원칙
1. /example-skills:frontend-design skills 를 사용해야 한다
2. shadcn 을 사용할 예정이고 white/dark 모드 전환이 가능해야 하고 width 변화에 따라 변하는 반응형으로 만들어야 합니다.
3. 브라우저 검증이 필요하면 playwright plugin 을 사용하고 localhost 로 접속하지 말고 https://embed.cunlim.dev 로 접속해주세요.
4. `/` 랜딩 페이지의 스타일은 height 는 길게 필요 없고 깔끔함 70%, 특수효과 + 개발자스러운 스타일 30% 가중치를 고려해주세요.
5. `/embed` 기술 시연 페이지 중 검색 키워드 및 결과 도출 데이터를 사용자가 한눈에 파악할 수 있도록 볼드(Bold) 및 수치화하여 하이라이팅한다.
