# PRD: AI 기반 다국어 카테고리 추천 시스템

## 목표

사용자의 검색어나 텍스트를 분석하여 네이버 카테고리 체계 기준으로 최적의 카테고리를 코사인 유사도 검색으로 추천하는 다국어(한국어/중국어/영어) 포트폴리오 서비스.

## 사용자

- **비회원(게스트)**: 메인 페이지 접근, 검색/추천 기능 사용, 추천 개수 설정
- **회원**: 비회원 기능 + 카테고리 추가/수정/삭제 권한
- **관리자**: 전체 카테고리 관리, 일괄 번역/임베딩 파이프라인 트리거

## 핵심 기능

### Phase 1: 백엔드 — Laravel 비동기 텍스트 파이프라인
1. **번역 파이프라인**: 한국어 카테고리명 → `>` 분할 → `translation_cache` 조회/삽입 → Ollama `translategemma:4b` 번역 → 재조립 → `categories` 테이블 저장
2. **임베딩 파이프라인**: 번역 완료 후 언어별 `category_name` → Ollama `nomic-embed-text`로 벡터화 → `category_embeddings` 테이블 저장 (768차원 VECTOR)
3. **예외 처리**: 정규식 검증 + 최대 3회 재시도, 최종 실패 시 `failed_jobs` 테이블 기록
4. **Rate Limit 대응**: Redis Throttle 또는 Job 간 딜레이(sleep)로 429 우회
5. **중복 방지**: `Cache::lock()`으로 동일 언어+모델 조합의 동시 실행 차단

### Phase 2: 프론트엔드 — Next.js 실시간 UI
1. **실시간 Progress**: Laravel Echo + Reverb WebSocket 구독, 번역/임베딩 진행률 프로그레스 바
2. **언어 선택 UI**: 한국어/중국어/영어 토글, 임베딩 모델 선택 Select Box
3. **일괄 처리 트리거**: 관리자만 접근 가능한 배치 실행 버튼 (중복 실행 방지)

### Phase 3: 검색 — 언어별 pgvector 코사인 유사도 검색
1. **LIKE 검색**: 타겟 언어 필드에서 `LIKE '%키워드%'` 쿼리, 결과 키워드 **Bold** 처리
2. **AI 추천**: 입력 텍스트를 벡터화 → `category_embeddings`에서 코사인 유사도 상위 N개 반환
3. **임베딩 캐시**: 동일 `search_keyword` 재검색 시 `search_logs` 테이블의 임베딩 재사용
4. **결과 UI — 버튼 A**: 벡터 계산 과정 모달 (원시 벡터 앞뒤 일부 + `...` + 유사도 수치)
5. **결과 UI — 버튼 B**: 동적 계층형 Select Box 모달 (`A>B>C` 파싱 → 3단계 Select Box 실시간 연동)

### 부가 기능
6. **개별 카테고리 추가**: 한국어 카테고리명 단일 입력 → `category_code` 자동 생성 → 동일 Job 클래스로 백그라운드 번역/임베딩 자동 실행
7. **인증**: 이메일/비밀번호, Google, GitHub, Naver OAuth (Socialite)
8. **검색 이력**: `search_logs` 테이블에 기록, 재검색 제공

## MVP 제외 사항

- 비회원→회원 마이그레이션 (데이터 이동 로직)
- 카테고리 수정/삭제 UI (관리는 DB 직접)
- 카테고리 계층结构的 대시보드 UI
- 다중 임베딩 모델 비교 기능 (UI)
- 외부 벡터 DB (Pinecone, Qdrant) 연동
- AI 응답 품질 자동 평가/모니터링

## 디자인

### 방향
- **도구 같은 대시보드**: 마케팅 페이지가 아닌, 매일 쓰는 기능성 인터페이스
- **다크모드 고정**: 사용자 눈부심 방지, AI 서비스 특유의 깔끔한 인상
- **미니멀**: 불필요한 장식 제거, 기능이 곧 디자인

### 레이아웃
- 전체 너비: `max-w-5xl` 기준, 콘텐츠에 따라 `max-w-7xl`
- 정렬: 좌측 정렬 기본, 중앙 정렬 금지
- 간격: 구성 요소 간 `gap-3` ~ `gap-4`, 섹션 간 `space-y-8`
- 카드: `rounded-lg bg-[#141414] border border-neutral-800 p-6`
