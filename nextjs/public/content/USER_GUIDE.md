# 사용자 가이드

## 개요

CL Embed는 AI 기반 다국어 카테고리 추천 시스템입니다. 상품명이나 설명 텍스트를 입력하면 가장 적합한 카테고리를 한국어, 영어, 중국어로 동시에 추천합니다.

## 시작하기

### 웹 인터페이스

[기능시연 페이지](/embed)에서 바로 사용해볼 수 있습니다.

1. 검색창에 상품명이나 설명을 입력하세요 (예: "여름용 원피스", "블루투스 이어폰")
2. 대상 언어를 선택하세요 (ko / en / zh)
3. AI가 입력 텍스트와 가장 유사한 카테고리를 찾아 추천합니다

### 카테고리 탐색

기능시연 페이지 왼쪽 사이드바에서 전체 카테고리 트리를 탐색할 수 있습니다. 카테고리를 클릭하면 하위 카테고리가 펼쳐집니다.

### 접근 권한

| 기능 | 게스트 | 로그인 |
|---|---|---|
| 카테고리 조회/검색 | O | O |
| 카테고리 계층 탐색 | O | O |
| 추천 실행 | O | O |
| 카테고리 추가/삭제 | X | O |
| 번역/임베딩 실행 | X | O |

## API 연동

### 인증

API 사용을 위해서는 로그인 후 발급되는 Bearer 토큰이 필요합니다. [로그인 페이지](/login)에서 이메일 또는 OAuth(Google, GitHub, Naver)로 인증할 수 있습니다.

### 엔드포인트

#### 카테고리 추천

```
POST /api/recommend
Authorization: Bearer {token}
Content-Type: application/json

{
  "keyword": "검색할 텍스트",
  "lang": "ko",
  "per_page": 5
}
```

#### 카테고리 계층 조회

```
GET /api/categories/levels
Authorization: Bearer {token}
```

### 언어 코드

| 코드 | 언어 |
|---|---|
| `ko` | 한국어 |
| `en` | 영어 |
| `zh` | 중국어 |

## 지원 언어

카테고리 추천은 한국어(ko), 영어(en), 중국어(zh) 3개 언어를 지원합니다. 입력 텍스트의 언어에 관계없이 원하는 타겟 언어의 카테고리를 추천받을 수 있습니다.

## 마이페이지

[마이페이지](/mypage)에서 API key를 관리하고 사용량을 확인할 수 있습니다.

### API Key 관리

- 여러 개의 API key를 생성할 수 있습니다
- 각 key를 개별적으로 일시정지/활성화할 수 있습니다
- 불필요한 key는 삭제할 수 있습니다

### 사용량 확인

- 총 호출 횟수, 오늘 호출 횟수, 남은 무료 회수를 확인할 수 있습니다
- 기간별 호출 추이 차트를 볼 수 있습니다
- 최근 호출 이력을 확인할 수 있습니다

### 무료 회수

회원가입 시 100회 무료 호출이 제공됩니다. 무료 회수를 모두 사용하면 관리자에게 문의하세요.

## 외부 API (v1)

외부 시스템에서 카테고리 유사도 검색을 호출할 수 있습니다. 자세한 내용은 [API 문서](/content/api-v1.md)를 참조하세요.

### 엔드포인트

```
POST /api/v1/search
Authorization: Bearer cl_xxxxxxxxxxxxxxxxxxxx
```

### 주요 파라미터

| 파라미터 | 필수 | 설명 |
|----------|------|------|
| `text` | O | 유사도 검색어 |
| `target_language` | - | 대상 언어 (기본: ko) |
| `page` | - | 페이지 번호 (기본: 1) |
| `per_page` | - | 페이지당 결과 수 (기본: 20, 최대: 50) |
